# Contract-Billing-System
#Introduction
Conbill is a web-based system to manage contracts and billing processes from an IT Department perspective. It is built for small to medium-sized businesses/organizations. It is built with HTML, CSS (Frontend), PHP (Backend), XAMPP (Apache + MySQL)(Server), and MySQL (Database).

# System Objectives
-Manage client information efficiently
-Store and track contracts
-Generate and manage invoices
-Record and track payments

# Core Modules
-Client Management
-Contract Management
-Invoice Management
-User Authentication

# Database Structure
-Clients
-Contracts
-Invoices
-Invoice items

# System benefits
- Organized data storage
- Automated invoice tracking
- Reduced manual errors
- Improved financial stability
