# Contract-Billing-System
# Introduction
This is my internship project, which involves 2 interns one works on the UI and I worked on the database. Conbill is a web-based system to manage contracts and billing processes from an IT Department perspective. It is built for small to medium-sized businesses/organizations. It is built with HTML, CSS (Frontend), PHP (Backend), XAMPP (Apache + MySQL)(Server), and MySQL (Database).

# System Objectives
- Manage client information efficiently
- Store and track contracts
- Generate and manage invoices
- Record and track payments

# Features
- Client and company management
- Contract tracking with renewal dates
- Invoice generation and tracking
- Itemized billing system

# Database Structure
- Clients
- Contracts
- Invoices
- Invoice items

# System benefits
- Organized data storage
- Automated invoice tracking
- Reduced manual errors
- Improved financial stability

User authentication (admin, staff, user)

# How to Use
1. Import the .sql file into MySQL/MariaDB
2. Run the system locally using PHP/XAMPP
